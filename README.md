# cylinder-quine

A quine that creates a 3d model of itself

## Install

```sh
git clone https://github.com/donno2048/cylinder-quine
cd cylinder-quine
pip3 install -r requirements.txt
```

## Usage

```sh
python3 quine.py
```

You can use the version with the better style like so:

```sh
git apply quine.patch
python3 quine.py
```
